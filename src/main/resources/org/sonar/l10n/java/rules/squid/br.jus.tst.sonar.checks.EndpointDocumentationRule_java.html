This rule detects when a class defining a REST resource isn't documented

Noncompliant Code Example

@Path("clients")
public class ClientResource {

    @GET
    public Response findAll() {
        ...
    }

}

Compliant Solution

@Path("clients")
@Api("clients")
public class ClientResource {

    @GET
    @ApiOperation(
        value = "Returns all registerd clients",
        response = Client.class,
        responseContainer = "List",
        code = HttpStatus.SC_OK
    )
    public Response findAll() {
        ...
    }

}
